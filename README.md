Task 1

NUMBER GUESSING GAME
The system generates a random number from a given range, say 1 to 100
There is a simple GUI that a user can easily interact with.
The computer then tells if the entered number matches the guesses number or if it is higher/lower than the generated number.
The game continues under the user guessing the number.
Users have 2 rounds and 5 attempts each round
